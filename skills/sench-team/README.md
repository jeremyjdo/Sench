# /sench-team

Team and founders analysis skill for AI agents.

## Install

```bash
npx skills add https://github.com/jeremyjdo/Sench --skill sench-team
```

## Usage

```
/sench-team alan
/sench-team qonto
/sench-team pennylane
```

## Output

Single file: `{company}_team.md`

Contains:
- Synthesis of team composition
- Founder deep dives (background, education, previous experience)
- Employee list by department with LinkedIn URLs
- Team analysis (strengths, concerns, hiring signals)
