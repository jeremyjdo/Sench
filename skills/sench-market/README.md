# /sench-market

Market definition & opportunity mapping for venture design.

## What It Does

Creates comprehensive opportunity maps that serve as the top funnel for venture design. Defines the broadest possible market perimeter before any prioritization or filtering.

## Philosophy

**Scope broadly, filter later.**

- Prioritize coverage over early precision
- Reserve judgment on viability until scoping is complete
- Use value chains as exploration scaffolds
- Document assumptions explicitly
- Resist premature focus

## Usage

```
/sench-market pet insurance
/sench-market vertical farming
/sench-market regulatory reporting --fr
```

## Output

Single markdown file: `{market}_opportunity_map.md`

Includes:
- Market definition & boundaries
- Value chain mapping
- Opportunity map (visual grid)
- Segment deep dives
- Adjacent markets analysis
- Competitive landscape
- International benchmarks
- Trends & drivers
- Customer insights (JTBD)
- Key questions for exploration

## Key Questions Answered

- What could reasonably belong to this market?
- What problems or use cases exist that aren't obvious?
- What adjacent or emerging spaces are relevant?
- What exists in other countries that doesn't exist locally?
- What parts of the value chain are overlooked?
- What would we regret not exploring later?

## Requirements

- WebSearch
- WebFetch

No API keys required.
